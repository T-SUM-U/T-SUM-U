説明

惑星を生成して恒星系を作るゲームです。恒星系の完成度（美しさ）で得点が決まります。
  • 注意
    • 背景の内、名前に「Star Nest」と付いているものは使用中のPCに多大なる負荷を与えます。
    • 背景に限らず、このゲームはPCに与える負荷が大きめです。
    • 惑星の軌道が双曲線軌道の場合の軌道離心率の計算は正確ではありません。近日中に修正したいと考えています。

    この作品はUnityインターハイ2019参加作品です。
